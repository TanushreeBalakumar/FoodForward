# 🍱 Food Forward

Food Forward is a full-stack web application designed to connect food donors with people in need. The platform helps reduce food waste by enabling donors to share excess food and allowing receivers to search for available donations nearby through a secure and user-friendly system.

### 👥 User Authentication
- Receiver Registration & Login
- Secure access to food search functionality
- Restricted pages for authenticated users

### 🍛 Food Donation System
- Donors can submit food donation details
- Food availability tracking
- Organized donation records

### 🔍 Search Functionality
- Receivers can search for available food donations
- Login required before accessing the search page

### 🛠️ Admin Dashboard
- View registered receiver details
- View donation details
- Track number of food donations by date
- Manage contact form submissions

### 🤖 AI Chatbot
- Integrated chatbot support for user interaction

### 🎨 Responsive UI
- Modern and responsive interface
- User-friendly design for better accessibility

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone <your-repository-link>
cd FoodForward
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Configure Database

* Create a MySQL database
* Import the required tables
* Update database credentials in your configuration file

### 4️⃣ Run the Server

Using Node:

```bash
node server.js
```

Using Nodemon:

```bash
nodemon server.js
```

or

```bash
npm run dev
```

### 5️⃣ Open in Browser

```bash
http://localhost:3000
```
